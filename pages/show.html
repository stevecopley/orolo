Showcase

I enjoy designing and creating resources for my teaching and for fun. You can find me on GitHub: @waimea-cpy for work stuff, and @stevecopley for personal projects.

Here are a few projects I've worked on...

Cryptography Demos [GH]

Simulations of historic and modern cryptography (Pigpen, Caesar, Vignere, Public-Key)

Binary Number Explorer [GH]

An interactive binary number explorer, displaying various representations of the value

Logic Circuit Simulator [GH]

Drag and drop logic circuit simulator using the HTML canvas.

Quizzes for Coding [GH]

Presents quizzes based on markdown files that can contain code snippets, flowcharts, etc.

Kotlin Kolour Library [GH]

Small Kotlin library to enable colour text output to the terminal.

IT Tips for Waimea College Staff [WWW]

Tips and tricks for Waimea College staff to use in their teaching. Note: geo-blocked to Aus/NZ.

WOPR Simulator [GH]

A re-creation of the WOPR terminal command-line interface from the 80s film, War Games.

Gist List Viewer [GH]

A viewer for a GitHub user's gists that makes them easy to browse, filter, sort, etc.

IGCSE ICT [WWW]

Notes and resources for Cambridge International Exams IGCSE ICT. Note: has not been updated for a long time!
